# CalculadoraDeGastos_py_funcional
1. Funções de Primeira Classe e Alta Ordem
Em Python, as funções são "cidadãs de primeira classe". Isso significa que você pode tratá-las como qualquer outra variável: passá-las como argumento para outras funções, retorná-las ou guardá-las em listas.
2. Funções Puras (Pure Functions)
Uma função é considerada pura se ela cumpre dois requisitos:

Mesmo input, mesmo output: Se você passar os mesmos argumentos, ela sempre retornará o mesmo resultado.

Sem efeitos colaterais: Ela não altera nada fora dela (não modifica variáveis globais, não altera listas passadas por referência, não escreve em arquivos, etc.).
3. Imutabilidade (Immutability)
No paradigma funcional puro, você não altera dados; você cria novos dados a partir dos antigos. 
4. Funções Anônimas (Expressões lambda)
As funções lambda são funções descartáveis e sem nome, escritas em apenas uma linha. Elas são muito usadas junto com ferramentas funcionais para operações rápidas.
